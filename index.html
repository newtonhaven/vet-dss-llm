Abstract

VetDSS is a decision support system designed to assist veterinarians in their daily practice.

Our system helps reduce workload, minimize diagnostic errors, and enhance the decision-making process by recommending possible diagnostic tests and treatments. By leveraging advanced algorithms and a comprehensive database, VetDSS provides accurate and timely suggestions to support veterinarians in delivering high-quality care.

VetDSS integrates seamlessly into veterinary practices, offering an intuitive interface and robust functionality. It is designed to be a valuable tool for both experienced veterinarians and those new to the field, ensuring that all users can benefit from its capabilities.



Performance

Using VetDSS, which leverages 4-bit quantization and the Mistral 7B LLM model, you can achieve more efficient and better performance in veterinary decision support.

Dataset

For testing, we used the following datasets: VetPetCare, Animal Disease, and data from the ASPCA. Additionally, VetDSS can be fed with self-collected data to enhance its performance.
